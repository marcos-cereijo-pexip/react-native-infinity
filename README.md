# @pexip/react-native-infinity

React Native library for Infinity.

## Installation

```sh
npm install @pexip/react-native-infinity
```

## Usage

```js
import { multiply } from '@pexip/react-native-infinity';

// ...

const result = await multiply(3, 7);
```
